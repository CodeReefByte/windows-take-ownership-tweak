Take Ownership Context Menu Tweaks for Windows 10 Pro

Files Included:
- TakeOwnership.reg       → Adds "Take Ownership" to the right-click menu for files and folders.
- RemoveTakeOwnership.reg → Removes the context menu entry.

Instructions:
1. Right-click TakeOwnership.reg and select "Merge" to add the option.
2. Right-click RemoveTakeOwnership.reg and select "Merge" to remove it later.
3. After merging, you can safely delete these .reg files.

Note: These changes require administrator privileges.
